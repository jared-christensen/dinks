## How to Sign Up (Membership Instructions)

### Option 1: Mobile App

1. Download CourtReserve App
2. Create an account
3. Search and select "Dinks Pickleball" from drop down menu
4. Select membership type and payment type

### Option 2: Web Browser

1. Visit www.courtreserve.com
2. Select login > "don't have an account" > "I'm a player"
3. Search and select "Dinks Pickleball"
4. Create account and select membership type
